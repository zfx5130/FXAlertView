# FXAlertView

###自定义弹出视图
 
 * 视图中间部分可更具自己需求自定义.
 * 弹出视图button,有三种形式
   1.纯文字按钮
   2.图片结合文字按钮
   3.图片按钮
   
####效果如下

![](https://github.com/zfx5130/FXAlertView/blob/master/images/demo.gif)
![](https://github.com/zfx5130/FXAlertView/blob/master/images/demo1.gif)

##安装
###1.CocoaPods

```
  1.在 Podfile 中添加 pod "FXAlertView"。
  2.执行 pod install
  3.导入 <FXAlertView.h>。

```
